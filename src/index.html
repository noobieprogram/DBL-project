Welcome! This is a data visualization tool, developed by students. It supports Newick format, so you can upload a file containing Newick dataset (.txt, .tre, .nw) and it will render four visualizations on your screen. All of the visualizations are interactive - if you click on "?" you can see the possible interactions. Enjoy!
